# simple-postgresql-high-avaliability

## Обзор решения

SPGHA представляет собой набор bash скриптов для поддержания репликационной пары PostgreSQL в рабочем состоянии.
Вместе со скриптами предполагается использование Keepalived для предоставления доступа к мастер-инстансу Postgres по VIP.

### Добавление systemd сервисов

Запуск скриптов осуществляется через systemd по

На узлах с **полновесным** контроллером spgha добавьте systemd сервис:
`vim /lib/systemd/system/spgha_active_node_controller.service`

```bash
[Unit]
Description=Manages high avaliability for PostgreSQL replication pair
Wants=network-online.target
After=network-online.target
StartLimitIntervalSec=3600
StartLimitBurst=2

[Service]
Type=simple
ExecStart=<путь до скрипта spgha>
KillMode=process
TimeoutStopSec=5
Restart=on-failure
RestartSec=5

[Install]
WantedBy=multi-user.target
```

На узлах с **голосующим** контроллером spgha добавьте systemd сервис:
`vim /lib/systemd/system/spgha_voter_node_controller.service`

```bash
[Unit]
Description=A lightweight voter for Simple PostgreSQL High Avaliability
Wants=network-online.target
After=network-online.target
StartLimitIntervalSec=3600
StartLimitBurst=2

[Service]
Type=simple
ExecStart=<путь до скрипта spgha>
KillMode=process
TimeoutStopSec=5
Restart=on-failure
RestartSec=5

[Install]
WantedBy=multi-user.target
```

После добавления новых сервисов перечитайте конфигурацию systemd
`systemctl daemon-reload`

### Подготовка keepalived

В процессе работы SPGHA опирается на keepalived для предотвращения подключения к СУБД в сценарии **split brain** и предоставления единой точки входа.
Поэтому одним из этапов подготовки является установка и конфигурация сервиса keepalived.

В данной установке механизм VRRP не применяется, а узел, на котором необходимо поднять VIP, определяется следующим скриптом:

```bash
#!/bin/bash

PG_ISREADY="<Путь до утилиты pg_isready>"
PSQL="<Путь до утилиты psql>"

if [[ -e /etc/keepalived/no_vip.block ]]; then
    exit 1
fi

if ! "$PG_ISREADY" -q -h 127.0.0.1 -p 5432 -U postgres; then
    exit 1
fi

is_slave=$("$PSQL" -h 127.0.0.1 -p 5432 -U postgres -tAc "SELECT pg_is_in_recovery();")
if [[ "$is_slave" = "t" ]]; then
    exit 1
else
    exit 0
fi
```
